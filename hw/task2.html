Lesson 16 - Homework - Task 2

2. Определение времени суток

Задача:

Напишите программу, которая принимает на вход час (целое число от 0 до 23) и определяет время суток:

  • 0-5: Ночь
  • 6-11: Утро
  • 12-17: День
  • 18-23: Вечер
  • Используйте if-else для определения времени суток.
  • Добавьте проверку корректности ввода (например, если пользователь ввёл число вне диапазона 0-23).
