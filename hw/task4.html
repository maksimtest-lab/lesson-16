Lesson 16 - Homework - Task 3

4. Оценка студента

Задача:

Напишите программу, которая принимает на вход оценку (0-100) и выводит уровень знаний студента:

  • 90-100: Отлично
  • 70-89: Хорошо
  • 50-69: Удовлетворительно
  • 0-49: Неудовлетворительно
  • Используйте if-else для определения уровня.
  • Добавьте обработку исключений с помощью try-catch, чтобы предотвратить ошибки при некорректном вводе (например, если вместо числа введён текст).
